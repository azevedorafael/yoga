# Our design-system

Following design guide-lines specification, we developed a React UI library antd that contains a set of high quality components that dictates Gympass interfaces.

## Getting Started

Our Design-system is dedicated to providing a good development experience for programmers and users.

### Installing

In order to install our design-system just use `yarn` or `npm` package manager

```
yarn add @gympass/yoga
```
