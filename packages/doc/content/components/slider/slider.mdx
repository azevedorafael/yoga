# Slider

## Import the Slider

```javascript
import { Slider } from '@gympass/yoga';
```

## Usage

```javascript reactLive=true
<Slider
  max={10}
  min={0}
  maxLabel={10}
  minLabel={0}
  tooltip={[
    {
      ribbon: 'My plan',
      title: 'Up to',
      description: 'U$ 99,90',
      step: 2,
      visible: true,
    },
  ]}
/>
```
