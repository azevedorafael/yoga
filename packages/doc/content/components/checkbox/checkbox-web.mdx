### Reference

Gympass `<Checkbox />` are used for a list of options where the user may select multiple options, including all or none.

### Usage

```javascript reactLive=true center=true
<section>
  <Checkbox
    style={{ width: 200, marginBottom: 40 }}
    id="checkbox-1"
    value="checkbox-value"
    label="Checkbox Unchecked"
  />

  <Checkbox
    checked
    style={{ width: 200 }}
    id="checkbox-1"
    value="checkbox-value"
    label="Checkbox Checked"
    helper="Checkbox Helper"
  />
</section>
```
