# AutoComplete

## Reference

It accepts unlimited number of suggstions, but it filters and only display the six
most matched options.

You can use `onSelect` prop to handle when user select a suggested option.

It inherit all the [Input](./input/default) props.

## Usage

```javascript state
render(() => {
  const [value, setValue] = useState('');

  return (
    <AutoComplete
      value={value}
      onChange={e => setValue(e.target.value)}
      onSelect={selected => setValue(selected)}
      onClean={cleanable => setValue(cleanable)}
      options={[
        'Madrid',
        'Milan',
        'SÃ£o Paulo',
        'New York',
        'Paris',
        'Buenos Aires',
      ]}
    />
  );
});
```
