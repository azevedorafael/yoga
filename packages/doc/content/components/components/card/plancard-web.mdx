### Reference

Gympass `<Card />` component can be used to display content related to a single subject.
The content can consist of multiple elements of varying types and sizes.

### Usage

```javascript reactLive=true center=true
<PlanCard
  style={{ width: 280 }}
  ribbon={{
    text: 'Label',
    variant: '',
  }}
>
  <PlanCard.Content
    title="Gympass Black for employees with Buddha Spa + Reebook included muitas outras coisas"
    price="U$ 99.90"
  />
  <p>This is a PlanCard component, you can use it to display a content.</p>
</PlanCard>
```
