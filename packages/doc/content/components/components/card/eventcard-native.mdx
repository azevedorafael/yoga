### Reference

Gympass `<EventCard />` is used to apply a container around a related
grouping of information about Gympass events.

### Usage

```javascript type=expo
<EventCard
  style={{ width: 250 }}
  event={{
    name: 'Yoga Class',
    place: 'Gympass - Live the Mission',
    time: '19 am',
  }}
  date={{
    day: '19',
    dayOfWeek: 'thu',
    month: 'dez',
  }}
/>
```
