# GymCard.CheckIn

### Reference

Gympass `<GymCard.CheckIn />` component can be used to display content related a
check-in.

### Usage

The `avatar` prop receives the same object as the `source` prop of the [Image](https://facebook.github.io/react-native/docs/images#source-as-an-object)
`react-native` component.
