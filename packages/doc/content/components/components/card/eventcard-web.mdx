### Reference

Gympass `<EventCard />` is used to apply a container around a related
grouping of information about Gympass events.

### Usage

```javascript reactLive=true center=true
<EventCard
  event={{
    name: 'Yoga Class',
    place: 'Gympass',
    time: '19 am',
  }}
  date={{
    day: '19',
    weekday: 'thu',
    month: 'dez',
  }}
/>
```
