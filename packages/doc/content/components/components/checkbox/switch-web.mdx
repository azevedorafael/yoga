### Reference

Gympass `<Checkbox.Switch />` component implements all our design guidelines such as default spacings, borders and color pallete tokens.
Switches are best used for communicating activation (e.g., on/off states), while checkboxes are best used for communicating selection (e.g., multiple table rows). Switches, unlike checkboxes, can't have an error state.

### Usage

```javascript reactLive=true center=true state=true
render(() => {
  const [checked, setChecked] = useState(false);
  const onChange = () => setChecked(!checked);

  return <Checkbox.Switch checked={checked} onChange={onChange} />;
});
```
