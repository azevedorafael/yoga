### Reference

Gympass `<Checkbox />` are used for a list of options where the user may select multiple options, including all or none.

### Usage

```javascript reactLive=true center=true state=true
render(() => {
  const [checked, toggleChecked] = useState(false);
  const onChange = () => toggleChecked(!checked);

  return (
    <Checkbox
      checked={checked}
      onChange={onChange}
      value="checkbox-value"
      label="Checkbox"
      helper="Checkbox Helper"
    />
  );
});
```
