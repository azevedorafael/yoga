### Reference

Gympass `<Button />` component implements all our design guidelines such as default spacings, borders and color pallete tokens.
You can use all differente compounds like `<Button />` and `<Button.Text />` combining with their props such as `small` or `inverted` so the button become the desired variation.

### Usage

```javascript
<Button.Outline>Find an activity</Button.Outline>
```
