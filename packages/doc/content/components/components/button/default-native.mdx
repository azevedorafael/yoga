### Reference

Gympass `<Button />` component implements all our design guidelines such as default spacings, borders and color pallete tokens.
You can use all differente compounds like `<Button.Outline />` and `<Button.Text />` combining with their props such as `small` or `inverted` so the button become the desired variation.
