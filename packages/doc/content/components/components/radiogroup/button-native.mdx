### Reference

`<RadioGroup.Button />` creates a button instead of a radio default looks, be sure to use `<RadioGroup.Button />` as `<RadioGroup />` children so it inherit default spacing, border and color pallete tokens.

### Usage

```javascript type=expo
<RadioGroup selectedValue={'now'}>
  <RadioGroup.Button value="24h">24 hours</RadioGroup.Button>
  <RadioGroup.Button value="now">Open now</RadioGroup.Button>
  <RadioGroup.Button value="sundays">At Sundays</RadioGroup.Button>
</RadioGroup>
```
