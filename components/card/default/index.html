    Card

    • Web
    • Native

    Reference

    Gympass <Card /> component can be used to display content related to a single subject. The content can consist of multiple elements of varying types and sizes.

    Usage

    Hello,

    This is a Card component, you can use it to display a content.

     import { Card } from '@gympass/yoga';
    <Card style={{ width: 500 }}>
      <h1>Hello,</h1>
      <p>This is a Card component, you can use it to display a content.</p>
    </Card>
